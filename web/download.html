一直想把程序放在网上，让网友下载，开始把程序放在csdn，三个月也没见一个下载。后来发现百度网盘很好。

公开链接：http://pan.baidu.com/s/1kTx7ZEn

开放下载的是两个程序，一个是hiwii语言的命令行版本，一个是hiwii语言的web版本。

命令行版本执行很简单，在有java环境后，执行> java -jar hiwii0.9xxx.jar

web版执行和命令行版近似，执行 > java -jar hserver1.jar

但web版包含了几个网页，执行后必须用浏览器访问，比如装在本机，则 http://localhost

web版采用了内嵌jetty的方法，因此不需要tomcat等容器即可执行。在web目录，可以通过文本编辑的方式放入任意jsp文件，已完全具备了jsp容器的功能。准备在hiwii基础上开发一种template language，即模板语言，使用户能够在文本中嵌入hiwii语言。
